description: The deterministic video primitives behind VidJutsu.

## Social video import

Import TikTok and Instagram video posts into stable, tenant-owned VidJutsu assets. Repeated imports reuse the same asset so retries and downstream jobs do not download the reference again.

## Scrape

Pull structured profile, post, comment, and search data from TikTok and Instagram without downloading media. Useful for research before you decide what to download.

## Watch

Use `POST /v1/watch` for prompted video or image understanding. Every watch request requires a prompt describing the evidence you need back from the model.

## Transcribe

Get a speech-to-text transcript with word-level timing for any video or audio URL.

## Compliance check

Run a TOS/compliance check against configurable rules before you publish. Rules are tenant-owned and can be read and updated through `GET/PUT /v1/check/rules`.

## Overlay and disclaimer

Burn a text overlay onto a video in TikTok-safe zones, or add a disclaimer card. Both return a durable output URL.

## Jobs and distribution

Media operations return a durable job you can poll for status, retries, and final artifacts. Distribution primitives (`/v1/distribution/clips`, `/v1/distribution/jobs`) hand finished assets off downstream.

## Interfaces

VidJutsu supports REST, a generated TypeScript SDK, the CLI, public agent skills, and an agent-readable auth manifest. A public MCP server is not configured today.
