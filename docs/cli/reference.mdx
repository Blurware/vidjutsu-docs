---
title: "vidjutsu reference"
description: "Manage references"
---

## Create

```bash
vidjutsu reference create --url https://instagram.com/p/xyz --platform instagram --tags "niche=fitness"
```

## List

```bash
vidjutsu reference list
vidjutsu reference list --platform instagram
```

## Get

```bash
vidjutsu reference get ref_abc123
```

## Edit

```bash
vidjutsu reference edit ref_abc123 --notes "Great hook technique"
```

## Delete

```bash
vidjutsu reference delete ref_abc123
```
