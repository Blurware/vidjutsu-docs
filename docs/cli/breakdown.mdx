title: "vidjutsu breakdown"

---

## Usage

```bash
vidjutsu breakdown --video-url <url> [--prompt <text>]
```

## Options

| Flag | Required | Description |
|------|----------|-------------|
| `--video-url` | Yes | URL of the video to analyze |
| `--prompt` | No | Guide the analysis focus (e.g. "Break down the hook and CTA") |

## Example

```bash
vidjutsu breakdown --video-url https://cdn.vidjutsu.ai/uploads/mc_.../video.mp4

Starting breakdown analysis...
Polling... pending
Polling... analyzing
Polling... completed

Hook:        "The one thing nobody tells you about creatine" (0-2s)
Format:      talking-head, single scene
Pacing:      fast — 3 cuts in 10s
Transitions: hard cut
CTA:         "link in bio" at 9s
Tags:        fitness, supplements
```
