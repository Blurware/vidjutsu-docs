title: "vidjutsu generate"
description: "Generate videos, images, and music"
---

## Videos

### Generate a video

```bash
vidjutsu generate --prompt "Street style lookbook, NYC summer" --aspect 9:16
```

### Options

| Flag | Type | Default | Description |
|------|------|---------|-------------|
| `--prompt` | string | required | Describe the video |
| `--model` | string | `veo-3.1` | `veo-3.1` or `veo-3.1-fast` |
| `--image` | string | — | Image URL for image-to-video mode |
| `--aspect` | string | — | `16:9`, `9:16`, or `1:1` |
| `--duration` | string | — | Duration in seconds |

### Example: image-to-video

```bash
vidjutsu generate \
  --prompt "Slow zoom into the product" \
  --image "https://example.com/product.jpg" \
  --aspect 9:16
```

## Images

### Generate an image

```bash
vidjutsu images generate --prompt "Minimalist product shot on white marble"
```

### Options

| Flag | Type | Default | Description |
|------|------|---------|-------------|
| `--prompt` | string | required | Describe the image |
| `--aspect` | string | — | Aspect ratio |
| `--resolution` | string | `1K` | `1K`, `2K`, or `4K` |
| `--format` | string | `png` | `png` or `jpg` |

### List images

```bash
vidjutsu images list
```

### Get image status

```bash
vidjutsu images get img_abc123
```

## Music

### Generate a track

```bash
vidjutsu music generate --prompt "Upbeat lo-fi for a fashion montage"
```

### Options

| Flag | Type | Default | Description |
|------|------|---------|-------------|
| `--prompt` | string | required | Describe the music |
| `--instrumental` | boolean | `true` | Instrumental only |
| `--title` | string | — | Song title (custom mode) |
| `--style` | string | — | Style tags (custom mode) |

### List tracks

```bash
vidjutsu music list
```

### Get track status

```bash
vidjutsu music get trk_abc123
```
