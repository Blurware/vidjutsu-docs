---
title: "vidjutsu watch"
description: "AI watches your video and answers what you ask."
---

## Usage

```bash
vidjutsu watch --mediaUrl <url> --prompt "..."
```

## Options

| Flag | Required | Description |
|------|----------|-------------|
| `--mediaUrl` | Yes | URL of the video or image to analyze |
| `--prompt` | Yes | Freeform prompt — ask anything about the video |

## Example

```bash
vidjutsu watch --mediaUrl https://cdn.vidjutsu.ai/uploads/mc_.../video.mp4 --prompt "Return JSON with a hookScore and evidence"

Watching video...

{
  "response": {
    "hookScore": 8,
    "evidence": ["Direct address begins in the first 1.2 seconds"]
  }
}
```

## Rate limit

Counts toward the daily `watch` limit (50/day). Resets at 00:00 UTC; exceeding it returns HTTP 429.
