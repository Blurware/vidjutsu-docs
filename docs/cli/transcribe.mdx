---
title: "vidjutsu transcribe"
description: "Speech-to-text with word-level timing."
---

## Usage

```bash
vidjutsu transcribe --mediaUrl <url>
```

## Options

| Flag | Required | Description |
|------|----------|-------------|
| `--mediaUrl` | Yes | URL of the video or audio to transcribe |

## Example

```bash
vidjutsu transcribe --mediaUrl https://cdn.vidjutsu.ai/uploads/mc_.../video.mp4

{
  "transcript": "The one thing nobody tells you",
  "words": [
    { "word": "The", "start": 0.0, "end": 0.82 },
    { "word": "one", "start": 0.82, "end": 1.1 }
  ],
  "language": "en",
  "duration": 14.2
}
```

## Rate limit

Counts toward the daily `transcribe` limit (30/day). Resets at 00:00 UTC; exceeding it returns HTTP 429.
