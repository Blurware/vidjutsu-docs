## Usage

```bash
vidjutsu score --video-url <url> --niche <keyword> [--type <format>]
```

## Options

| Flag | Required | Description |
|------|----------|-------------|
| `--video-url` | Yes | URL of the video to score |
| `--niche` | Yes | Target niche keyword (e.g. "fitness", "skincare") |
| `--type` | No | Content format (e.g. "talking-head", "b-roll") |
