description: "Score video quality from the command line."
---

## Usage

```bash
vidjutsu critic --video-url <url> [--context <text>]
```

## Options

| Flag | Required | Description |
|------|----------|-------------|
| `--video-url` | Yes | URL of the video to analyze |
| `--context` | No | Additional context for scoring (e.g. "Instagram reel, talking head") |

## Example

```bash
vidjutsu critic --video-url https://cdn.vidjutsu.ai/uploads/mc_.../video.mp4

Score:    9/10
Verdict:  Flawless. No AI artifacts or other issues detected.
Issues:   none
```
