---
title: "vidjutsu account"
description: "Create, edit, and manage account records"
---

## Create an account

Create an account record.

```bash
vidjutsu account create --platform instagram --name "nyc_streetstyle"
```

### Options

| Flag | Type | Description |
|------|------|-------------|
| `--platform` | string | Platform name (required) |
| `--name` | string | Account name (required) |
| `--niche` | string | Primary niche |
| `--handle` | string | Social media handle |
| `--bio` | string | Profile biography |
| `--pfp` | string | Profile picture URL |
| `--linkInBio` | string | Link in bio URL |
| `--tags` | string | Comma-separated key-value tags, e.g. `brand=acme,region=northeast` |

**Rate limit:** unmetered — account operations have no daily limit.

## Edit an account

Update an account record:

```bash
vidjutsu account edit acc_8k2x --bio "Updated bio" --linkInBio "https://newshop.com"
```

## List accounts

```bash
vidjutsu account list
```

## Get account details

```bash
vidjutsu account list --id acc_8k2x
```

## Delete an account

```bash
vidjutsu account delete acc_8k2x
```
