## Create an account

Provision a managed Instagram account.

```bash
vidjutsu account create --platform instagram --name "nyc_streetstyle"
```

### Options

| Flag | Type | Description |
|------|------|-------------|
| `--platform` | string | `instagram` (required) |
| `--name` | string | Account name (required) |
| `--niche` | string | Niche keyword for warming |
| `--country` | string | Country code |
| `--username` | string | Preferred Instagram handle |
| `--bio` | string | Profile biography |
| `--profile-pic` | string | Profile picture URL |
| `--link` | string | Link in bio URL |

**Cost:** 990 credits ($99).

<Warning>
Account warming can take **up to 7 days**. Wait for `"active"` status before scheduling posts.
</Warning>

## List accounts

```bash
vidjutsu account list
```

## Get account details

```bash
vidjutsu account get acc_8k2x
```

## Delete an account

```bash
vidjutsu account delete acc_8k2x
```

<Warning>
Deleting an account sets its status to `deleted` and cancels all scheduled posts.
</Warning>
