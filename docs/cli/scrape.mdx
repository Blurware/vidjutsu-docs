---
title: "vidjutsu scrape"
description: "Import TikTok and Instagram data and reference videos."
---

```bash
vidjutsu scrape <subcommand> [flags]
```

Use the `scrape` commands to discover public TikTok or Instagram references and their metadata. Scrape returns the provider's raw source media URLs and never stages media. To turn one video into a durable VidJutsu-hosted file, use the single-video download command, one video per call.

```bash
vidjutsu scrape tiktok-video \
  --url "https://www.tiktok.com/@creator/video/123456789"
```

Supported subcommands:

- TikTok: `tiktok-profile`, `tiktok-profile-videos`, `tiktok-video`, `tiktok-video-transcript`, `tiktok-video-comments`, `tiktok-search-users`, `tiktok-trending`
- Instagram: `instagram-profile`, `instagram-user-posts`, `instagram-post`, `instagram-post-comments`, `instagram-user-reels`

X, YouTube, and ad-library commands are not supported.
