---
title: "vidjutsu post"
description: "Create, edit, and manage post records"
---

## Create a post

Create a post record. The account ID is optional.

```bash
vidjutsu post create --videoId vid_q9f3 --caption "Summer fits"
```

### Options

| Flag | Type | Description |
|------|------|-------------|
| `--accountId` | string | Account ID to link to |
| `--videoId` | string | Video ID |
| `--mediaUrl` | string | Video URL |
| `--caption` | string | Post caption |
| `--brief` | string | Content brief or notes |
| `--tags` | string | Comma-separated key-value tags, e.g. `batch=week-12,format=talking-head` |

**Rate limit:** unmetered.

## Edit a post

Update a post record:

```bash
vidjutsu post edit post_m2v1 --caption "Updated caption" --mediaUrl "https://..."
```

## List posts

```bash
vidjutsu post list
```

Filter by account:

```bash
vidjutsu post list --accountId acc_8k2x
```

## Get post details

```bash
vidjutsu post list --id post_m2v1
```

## Delete a post

```bash
vidjutsu post delete post_m2v1
```
