## Subscription

VidJutsu requires an active **$99/month subscription** to use the API. Your subscription includes:

- **1,000 credits/month** (reset each billing cycle)
- Access to all intelligence endpoints (watch, extract, transcribe, check, overlay)
- Unlimited free storage operations (accounts, posts, assets, references, upload)

## How billing works

- **Monthly recurring** via Stripe. You're billed on the same date each month.
- **1,000 credits included** with your subscription. They reset at each billing cycle.
- **Additional credits** available at **$0.10/credit** anytime via `POST /v1/credits`.
- **No overage charges** — when you run out of credits, calls return a 402 error until you top up.

## Credit costs

| Operation | Endpoint | Credits |
|-----------|----------|---------|
| Watch | `POST /v1/watch` | 10 |
| Extract | `POST /v1/extract` | 5 |
| Transcribe | `POST /v1/transcribe` | 10 |
| Check | `POST /v1/check` | 5 |
| Overlay | `POST /v1/overlay` | 5 |

**Free operations** (0 credits): accounts, posts, assets, references, uploads, balance checks, check rules.

## Buy more credits

Top up at $0.10/credit when you need more than your included 1,000:

```bash
curl -X POST https://api.vidjutsu.ai/v1/credits \
  -H "Content-Type: application/json" \
  -d '{ "credits": 500, "email": "<your_email>" }'
```

Open the returned checkout URL to pay. Credits are added immediately.

## Cancel your subscription

Cancel anytime through the Stripe billing portal. Your existing credits remain usable, but API access ends when the current billing period expires.

## Get pricing programmatically

```bash
curl https://api.vidjutsu.ai/v1/pricing
```
