## Pricing model

VidJutsu runs on a **$99/month subscription** that includes **1,000 credits**. Intelligence calls burn credits; everything else is free.

Need more? Additional credits are available at **$0.10/credit** via Stripe checkout — buy as many as you need, any time.

## Credit costs

| Operation | Endpoint | Credits |
|-----------|----------|---------|
| Watch | `POST /v1/watch` | 10 |
| Extract | `POST /v1/extract` | 5 |
| Transcribe | `POST /v1/transcribe` | 10 |
| Check | `POST /v1/check` | 5 |

**Free operations** (0 credits): accounts, posts, uploads, list/read endpoints, pricing, balance checks, check/rules management.

## Checking your balance

```bash
curl https://api.vidjutsu.ai/v1/balance \
  -H "X-Api-Key: vj_live_abc123..."
```

```json
{
  "credits": 84,
  "clientId": "mc_xyz789"
}
```

## Buying additional credits

If you exhaust your included 1,000 credits, top up at $0.10/credit:

```bash
curl -X POST https://api.vidjutsu.ai/v1/api_keys \
  -H "Content-Type: application/json" \
  -d '{ "credits": 100, "email": "you@example.com" }'
```

This opens a Stripe checkout session. Credits are added immediately after payment.

## Getting pricing programmatically

```bash
curl https://api.vidjutsu.ai/v1/pricing
```
