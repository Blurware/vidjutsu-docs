## Pricing model

**$99 per managed Instagram account** (one-time). Usage-based pricing for all other operations. Subscription required ($59/mo includes 100 credits).

## How credits work

Every API operation costs a fixed number of credits at $0.10 each. Credits are deducted immediately when a request is accepted.

## Credit costs

| Operation | Endpoint | Credits | Cost |
|-----------|----------|---------|------|
| Account creation | `POST /v1/accounts` | 990 | $99.00 |
| Post scheduling | `POST /v1/posts` | 36 | $3.60 |
| Media analysis | `POST /v1/analyze` | 10 | $1.00 |
| Scraping | `POST /v1/scrape` | 10 | $1.00 |

**Free operations** (0 credits): All list/read endpoints, pricing, balance checks, campaign estimates, analytics, drafts, uploads.

## Account limit

Each user can create a maximum of **3 managed accounts**. This limit applies to both individual account creation and campaigns.

## Campaign pricing

Campaigns bundle account creation and post slots:

| Item | Credits | Cost |
|------|---------|------|
| Instagram account | 990 | $99.00 |
| Post slot (per post) | 36 | $3.60 |

Use `POST /v1/campaigns/estimate` to preview costs before creating a campaign.

## Checking your balance

```bash
curl https://api.vidjutsu.ai/v1/balance \
  -H "X-Api-Key: vj_prod_abc123..."
```

```json
{
  "credits": 84,
  "clientId": "mc_xyz789"
}
```

## Buying credits

```bash
curl -X POST https://api.vidjutsu.ai/v1/api_keys \
  -H "Content-Type: application/json" \
  -d '{ "credits": 100, "email": "you@example.com" }'
```

Pay with any card via Stripe. Credits are added immediately after payment.

## Subscription

$59/month includes 100 credits. Required for API access.

```bash
curl -X POST https://api.vidjutsu.ai/v1/subscriptions \
  -H "Content-Type: application/json" \
  -d '{ "email": "you@example.com" }'
```

## Getting pricing programmatically
