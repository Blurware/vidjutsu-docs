## Pricing model

**$99 per managed Instagram account** (one-time). Usage-based pricing for all other operations. No subscription required.

## How credits work

Every API operation costs a fixed number of credits at $0.75 each. Credits are deducted immediately when a request is accepted.

## Credit costs

| Operation | Endpoint | Credits | Cost |
|-----------|----------|---------|------|
| Account creation | `POST /v1/accounts` | 32 | $24.00 |
| Post scheduling | `POST /v1/posts` | 3 | $2.25 |
| Media analysis | `POST /v1/analyze` | 1 | $0.75 |
| Scraping | `POST /v1/scrape` | 1 | $0.75 |

**Free operations** (0 credits): All list/read endpoints, pricing, balance checks, campaign estimates, analytics.

## Account limit

Each user can create a maximum of **3 managed accounts**. This limit applies to both individual account creation and campaigns.

## Campaign pricing

Campaigns bundle account creation and video slots:

| Item | Credits | Cost |
|------|---------|------|
| Instagram account | 72 | $54.00 |
| Video slot (per post) | 2 | $1.50 |
| Edit slot (add-on, per post) | 3 | $2.25 |
| Comment moderation (add-on, per account) | 25 | $18.75 |

Use `POST /v1/campaigns/estimate` to preview costs before creating a campaign.

## Checking your balance

```bash
curl https://api.vidjutsu.ai/v1/balance \
  -H "X-Api-Key: vj_live_abc123..."
```

```json
{
  "credits": 84,
  "clientId": "cl_xyz789"
}
```

## Buying credits

```bash
curl -X POST https://api.vidjutsu.ai/v1/api_keys \
  -H "Content-Type: application/json" \
  -d '{ "credits": 100, "email": "you@example.com" }'
```

Pay with any card via Stripe. Credits are added immediately after payment.

## Getting pricing programmatically

```bash
curl https://api.vidjutsu.ai/v1/pricing
```

