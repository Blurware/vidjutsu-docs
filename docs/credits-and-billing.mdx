## Pricing model

Intelligence endpoints cost credits. Storage, CRUD, and billing endpoints are free.

## Credit costs

| Operation | Endpoint | Credits | Cost |
|-----------|----------|---------|------|
| Critic | `POST /v1/critic` | 10 | $1.00 |
| Breakdown | `POST /v1/breakdown` | 10 | $1.00 |
| Score | `POST /v1/score` | 10 | $1.00 |

**Free operations** (0 credits): accounts, posts, uploads, list/read endpoints, pricing, balance checks, breakdown polling.

## Checking your balance

```bash
curl https://api.vidjutsu.ai/v1/balance \
  -H "X-Api-Key: vj_live_abc123..."
```

```json
{
  "credits": 84,
  "clientId": "mc_xyz789"
}
```

## Buying credits

```bash
curl -X POST https://api.vidjutsu.ai/v1/api_keys \
  -H "Content-Type: application/json" \
  -d '{ "credits": 100, "email": "you@example.com" }'
```

Pay with any card via Stripe. Credits are added immediately after payment.

## Getting pricing programmatically

```bash
curl https://api.vidjutsu.ai/v1/pricing
```

```json
{
  "costs": {
    "critic": 10,
    "breakdown": 10,
    "score": 10,
    "accounts.create": 0,
    "posts.create": 0,
    "upload": 0
  }
}
```
