## Subscription

VidJutsu requires an active **$99/month subscription** to use the API. Access covers:

- Clone generation and its supporting character, check, and download endpoints
- Watch, extract, transcribe, check, compliance, overlay, and disclaimer endpoints
- Scrape and discovery endpoints
- Unmetered storage operations (accounts, posts, assets, references, projects, and upload)

## How billing works

- **Monthly recurring** via Stripe. You're billed on the same date each month.
- **Flat rate** — no per-call charges, no metered billing, no top-ups.
- **Daily rate limits** per endpoint (see below). Reset at 00:00 UTC.
- When you hit a limit, calls return **HTTP 429** with a `retryAfter` hint.

## Daily rate limits

Each row below is an independent bucket unless it is explicitly labeled shared. Hitting the `watch` limit does not affect `compliance`, while clone steps consume the same clone-admission bucket.

| Endpoint | Calls / day |
|----------|-------------|
| `POST /v1/watch` | 50 |
| `POST /v1/extract` | 100 |
| `POST /v1/transcribe` | 30 |
| `POST /v1/check` | 100 |
| `POST /v1/compliance/video` | 20 |
| `POST /v1/compliance/prompt` | 100 |
| `POST /v1/overlay` | 50 |
| Clone admission group (`clones/check`, characters, starting image, clone video) | 50 shared |
| Scrape and single-video download group | 500 shared |

Accounts, posts, assets, references, projects, uploads, and read-only endpoints are unmetered.

## Cancel your subscription

Cancel anytime through the Stripe billing portal. API access ends when the current billing period expires.

## Get pricing programmatically

```bash
curl https://api.vidjutsu.ai/v1/pricing
```

## Media limits

Limits depend on the operation. Clone-check, clone-video, and video watch inputs are capped at 15 seconds. Extract accepts videos up to 5 minutes. Binary uploads accept videos up to 32MB, images up to 10MB, and audio up to 25MB. Remote-media endpoints may apply their own content-type and fetch-size checks; consult the generated API reference for the exact request.
