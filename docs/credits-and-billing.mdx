## Pricing model

**Draft-first.** Creating resources (accounts, posts) is always free. Credits are only charged when you execute a command endpoint to provision or schedule.

Subscription required ($59/mo includes 100 credits).

## How credits work

Every billable operation costs a fixed number of credits at $0.10 each. Credits are deducted only at command endpoints — never at creation time.

## Credit costs

| Operation | Endpoint | Credits | Cost |
|-----------|----------|---------|------|
| Create draft account | `POST /v1/accounts` | 0 | Free |
| Provision account | `POST /v1/accounts/provision` | 990 | $99.00 |
| Create draft post | `POST /v1/posts` | 0 | Free |
| Schedule post | `POST /v1/posts/schedule` | 36 | $3.60 |
| Edit draft account | `PUT /v1/accounts?id={id}` | 0 | Free |
| Edit draft post | `PUT /v1/posts?id={id}` | 0 | Free |
| Media analysis | `POST /v1/analyze` | 10 | $1.00 |
| Scraping | `POST /v1/scrape` | 10 | $1.00 |

**Free operations** (0 credits): All list/read endpoints, pricing, balance checks, cost estimates, analytics, drafts, uploads, edits to drafts.

## Estimate costs

Use the stateless cost calculator before committing:

```bash
curl -X POST https://api.vidjutsu.ai/v1/estimate \
  -H "Content-Type: application/json" \
  -d '{
    "accounts": 3,
    "posts": 6,
    "platform": "instagram"
  }'
```

```json
{
  "credits": 3186,
  "breakdown": [
    { "label": "3 instagram accounts", "amount": 2970 },
    { "label": "6 post slots", "amount": 216 }
  ]
}
```

## Account limit

Each user can create a maximum of **3 managed accounts**. This limit applies to provisioned accounts (drafts do not count).

## Campaign pricing (legacy)

Campaigns bundle account creation and post slots. They still work but the recommended flow is draft resources + tags + command endpoints.

| Item | Credits | Cost |
|------|---------|------|
| Instagram account | 990 | $99.00 |
| Post slot (per post) | 36 | $3.60 |

## Buying credits

```bash
curl -X POST https://api.vidjutsu.ai/v1/api_keys \
  -H "Content-Type: application/json" \
  -d '{ "credits": 100, "email": "you@example.com" }'
```

Pay with any card via Stripe. Credits are added immediately after payment.

## Subscription

$59/month includes 100 credits. Required for API access.

```bash
curl -X POST https://api.vidjutsu.ai/v1/subscriptions \
  -H "Content-Type: application/json" \
  -d '{ "email": "you@example.com" }'
```
