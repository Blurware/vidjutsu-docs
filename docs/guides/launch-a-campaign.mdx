---
title: "Launch a Campaign"
description: "End-to-end guide: draft resources, tag them, provision and schedule"
---

There are two ways to run a campaign on VidJutsu:

1. **Draft-first (recommended)** — Create draft accounts and posts individually, tag them for organization, then provision and schedule with command endpoints.
2. **Campaign endpoints (legacy)** — Bundle account + post creation into a single campaign object. Still works but less flexible.

## Recommended: Draft-first workflow

### Step 1: Estimate costs

Preview how many credits you'll need:

```bash
curl -X POST https://api.vidjutsu.ai/v1/estimate \
  -H "Content-Type: application/json" \
  -d '{
    "accounts": 3,
    "posts": 6,
    "platform": "instagram"
  }'
```

```json
{
  "credits": 3186,
  "breakdown": {
    "accounts": { "count": 3, "perUnit": 990, "total": 2970 },
    "posts": { "count": 6, "perUnit": 36, "total": 216 }
  }
}
```

### Step 2: Create draft accounts (free)

```bash
curl -X POST https://api.vidjutsu.ai/v1/accounts \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: vj_live_abc123..." \
  -d '{
    "platform": "instagram",
    "name": "style_nyc",
    "username": "style_nyc",
    "bio": "NYC street style",
    "linkInBio": "https://myshop.com",
    "tags": [
      { "key": "campaign", "value": "nyc-summer" }
    ]
  }'
```

Repeat for each account. All drafts are free.

### Step 3: Provision accounts (990 credits each)

```bash
curl -X POST https://api.vidjutsu.ai/v1/accounts/provision \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: vj_live_abc123..." \
  -d '{
    "accountId": "acc_xxx1",
    "country": "US",
    "niche": "street fashion"
  }'
```

Returns `202 Accepted`. Repeat for each account.

<Warning>
Account warming can take **up to 7 days**. Provision accounts early.
</Warning>

### Step 4: Create draft posts (free)

```bash
curl -X POST https://api.vidjutsu.ai/v1/posts \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: vj_live_abc123..." \
  -d '{
    "accountId": "acc_xxx1",
    "videoUrl": "https://cdn.example.com/video1.mp4",
    "caption": "Summer style #fashion",
    "tags": [
      { "key": "campaign", "value": "nyc-summer" }
    ]
  }'
```

### Step 5: Schedule posts (36 credits each)

Once accounts are `"active"`:

```bash
curl -X POST https://api.vidjutsu.ai/v1/posts/schedule \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: vj_live_abc123..." \
  -d '{ "postId": "post_xxx1" }'
```

### Step 6: Monitor with tags

Filter all resources by campaign tag:

```bash
# List all accounts for this campaign
curl "https://api.vidjutsu.ai/v1/accounts?tag.campaign=nyc-summer" \
  -H "X-Api-Key: vj_live_abc123..."

# List all posts for this campaign
curl "https://api.vidjutsu.ai/v1/posts?tag.campaign=nyc-summer" \
  -H "X-Api-Key: vj_live_abc123..."
```

## Legacy: Campaign endpoints

Campaign endpoints bundle account creation and post slots into a single workflow. They still work but are less flexible than draft-first.

### Step 1: Estimate cost

```bash
curl -X POST https://api.vidjutsu.ai/v1/campaigns/estimate \
  -H "Content-Type: application/json" \
  -d '{
    "platform": "instagram",
    "accountCount": 3,
    "postsPerAccount": 2
  }'
```

### Step 2: Create the campaign

```bash
curl -X POST https://api.vidjutsu.ai/v1/campaigns \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: vj_live_abc123..." \
  -d '{
    "platform": "instagram",
    "accountCount": 3,
    "postsPerAccount": 2,
    "name": "NYC Summer",
    "niche": "street fashion",
    "accountProfiles": [
      { "username": "style_nyc", "biography": "NYC street style", "linkInBio": "https://myshop.com" },
      { "username": "fits_daily", "biography": "Daily outfit inspo" },
      { "username": "drip_la", "biography": "LA drip" }
    ]
  }'
```

### Step 3: Execute the campaign

```bash
curl -X POST https://api.vidjutsu.ai/v1/campaigns/execute \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: vj_live_abc123..." \
  -d '{ "campaignId": "cmp_abc123" }'
```

### Campaign lifecycle

| Status | What's happening |
|--------|-----------------|
| `draft` | Campaign created, credits reserved. Can be cancelled for full refund. |
| `executing` | Ordering accounts from TokPortal. |
| `provisioning` | Accounts are being set up on the platform. |
| `warming` | Accounts exist and are warming up with organic activity. |
| `ready` | All accounts are active and ready to post. |
| `failed` | Something went wrong. Check `errorMessage`. |
| `cancelled` | Campaign was cancelled, credits refunded. |

### Cancel a campaign

Cancel a `draft` or `executing` campaign to get a full credit refund:

```bash
curl -X POST https://api.vidjutsu.ai/v1/campaigns/cancel \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: vj_live_abc123..." \
  -d '{ "campaignId": "cmp_abc123" }'
```
