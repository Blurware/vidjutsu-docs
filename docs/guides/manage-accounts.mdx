VidJutsu provisions fully managed shortform accounts on Instagram. Create draft accounts for free, configure their profiles, then provision when you're ready to pay.

## What are managed accounts?

Managed accounts are purpose-built social media profiles for campaigns:

- **Theme pages** — Curated content around a niche (fashion, travel, fitness, tech)
- **Fan accounts** — Fan pages for artists, athletes, shows, or brands
- **Clip pages** — Compilation and highlight reels (sports clips, movie scenes, podcast cuts)
- **Promo pages** — Dedicated accounts that drive traffic to your link, shop, or digital product

You choose the platform and name. We provision the account, warm it up with organic-looking activity, and maintain it.

## Draft-first workflow

Account creation is now a two-step process:

1. **Create a draft** (free) — Set up the account profile without spending credits.
2. **Provision** (990 credits) — Charge credits and provision the account via TokPortal.

This lets you build and review your account setup before committing any credits.

## Step 1: Create a draft account

<CodeGroup>
```bash CLI
vidjutsu account create --platform instagram --name "nyc_streetstyle"
```

```bash API
curl -X POST https://api.vidjutsu.ai/v1/accounts \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: vj_live_abc123..." \
  -d '{
    "platform": "instagram",
    "name": "nyc_streetstyle",
    "username": "nyc_streetstyle",
    "bio": "NYC street style and fits",
    "linkInBio": "https://myshop.com",
    "tags": [
      { "key": "campaign", "value": "summer-launch" },
      { "key": "region", "value": "northeast" }
    ]
  }'
```
</CodeGroup>

**Cost:** 0 credits (free).

```json
{
  "id": "acc_8k2x",
  "status": "draft",
  "creditsCharged": 0
}
```

## Step 2: Edit the draft (optional)

Update a draft account's profile before provisioning:

```bash
curl -X PUT "https://api.vidjutsu.ai/v1/accounts?id=acc_8k2x" \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: vj_live_abc123..." \
  -d '{
    "bio": "Updated bio text",
    "linkInBio": "https://newshop.com",
    "tags": [
      { "key": "campaign", "value": "summer-launch" },
      { "key": "region", "value": "northeast" },
      { "key": "priority", "value": "high" }
    ]
  }'
```

<Warning>
PUT only works on accounts in `"draft"` status. Once provisioned, the account profile is managed by TokPortal.
</Warning>

## Step 3: Provision the account

When you're ready, provision the account to start creation and warming:

```bash
curl -X POST https://api.vidjutsu.ai/v1/accounts/provision \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: vj_live_abc123..." \
  -d '{
    "accountId": "acc_8k2x",
    "country": "US",
    "niche": "street fashion"
  }'
```

**Cost:** 990 credits ($99.00).

```json
{
  "id": "acc_8k2x",
  "status": "creating",
  "creditsCharged": 990
}
```

The response returns `202 Accepted`. The account will move through `creating` to `active` as it warms up.

## How warming works

After an account is provisioned, it goes through a warming period before it's ready to post. During warming, the account builds organic-looking activity to establish credibility on the platform.

<Warning>
**Warming can take up to 7 days.** Don't schedule posts until an account reaches `"active"` status. Posting during warmup can trigger platform flags.
</Warning>

## Account lifecycle

| Status | Description |
|--------|-------------|
| `draft` | Profile created, no credits charged. Editable via PUT. |
| `creating` | Account provisioned via TokPortal, credits charged |
| `active` | Ready to post — schedule content now |
| `suspended` | Temporarily paused (platform action) |
| `deleted` | Account has been removed |

## Tags

Accounts support AWS-style key-value tags for organization:

```json
{
  "tags": [
    { "key": "campaign", "value": "summer-launch" },
    { "key": "region", "value": "northeast" }
  ]
}
```

Tags are set at creation and updatable via PUT on drafts. Filter accounts by tag on the list endpoint:

```bash
curl "https://api.vidjutsu.ai/v1/accounts?tag.campaign=summer-launch" \
  -H "X-Api-Key: vj_live_abc123..."
```

## Build a network

Most campaigns use multiple accounts to amplify reach. Draft them all first, then provision:

```bash
# Create drafts (free)
vidjutsu account create --platform instagram --name "style_la"
vidjutsu account create --platform instagram --name "fits_daily"
vidjutsu account create --platform instagram --name "drip_atl"

# Provision when ready (990 credits each)
vidjutsu account provision acc_xxx1 --niche "street fashion" --country US
vidjutsu account provision acc_xxx2 --niche "street fashion" --country US
vidjutsu account provision acc_xxx3 --niche "street fashion" --country US
```

Each account drives traffic to the same destination — your link, shop, or product.

## Best practices

- **Draft early, provision later.** Set up all your accounts as drafts, review profiles, then provision when ready.
- **Use tags for organization.** Tag accounts by campaign, region, or priority for easy filtering.
- **Order accounts early.** Warming takes up to 7 days, so provision accounts before you need them.
- **Use descriptive names.** Account names help you identify which account is which when scheduling posts.
- **Focus on Instagram** — the primary platform for shortform sales content.
- **Wait for `active` status.** Never schedule posts to accounts that are still warming.
