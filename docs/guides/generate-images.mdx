---
title: "Generate Images"
description: "Create images for thumbnails, posts, and campaigns"
---

Generate images for campaign thumbnails, social posts, or as input for image-to-video.

## Generate an image

<CodeGroup>
```bash CLI
vidjutsu images generate --prompt "Minimalist product shot on white marble"
```

```bash API
curl -X POST https://api.vidjutsu.ai/v1/images/generate \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: vj_live_abc123..." \
  -d '{
    "prompt": "Minimalist product shot on white marble"
  }'
```
</CodeGroup>

**Cost:** 4 credits.

## Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `prompt` | string | required | Describe the image you want |
| `imageInput` | string | — | Reference image URL for style transfer |
| `aspectRatio` | string | — | Aspect ratio |
| `resolution` | string | `1K` | `1K`, `2K`, or `4K` |
| `outputFormat` | string | `png` | `png` or `jpg` |

## Check status

<CodeGroup>
```bash CLI
vidjutsu images list
```

```bash API
curl https://api.vidjutsu.ai/v1/images \
  -H "X-Api-Key: vj_live_abc123..."
```
</CodeGroup>

When `status` is `"completed"`, `resultUrls` contains your image URLs.
