---

VidJutsu imports public TikTok and Instagram references for brand meme cloning. It does not provide ad-library, X, YouTube, Reddit, Google, or LinkedIn scraping.

Use `download_media: true` when the next stage needs a durable VidJutsu-hosted file. Then pass that asset or URL to clone check.

```bash
vidjutsu scrape instagram-post \
  --url "https://www.instagram.com/reel/example/" \
  --download-media
```

The supported operations are TikTok profile, profile videos, video, transcript, comments, user search, and trending; and Instagram profile, posts, reels, post, and comments. All share the `scrape` rate group.

## Continue with the clone workflow

1. Import a TikTok or Instagram reference.
2. Run clone check before paid generation.
3. Create or select a character.
4. Create a clean starting image.
5. Submit the video clone with an idempotency key.
6. Apply the overlay and poll the durable job until final QA completes.

See the [clone meme videos guide](/guides/clone-meme-videos) for the complete workflow.
