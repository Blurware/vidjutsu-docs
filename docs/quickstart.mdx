## Option A: Use the CLI

The fastest way to get started. Install the CLI and authenticate in one step.

```bash
curl -fsSL https://vidjutsu.ai/install.sh | bash
```

Then authenticate with your card:

```bash
vidjutsu auth --card tok_live_your_card_token
```

Run a critic check:

```bash
vidjutsu critic --media-url "https://cdn.example.com/video.mp4" --media-type video
```

See the [CLI reference](/cli/index) for all commands.

## Option B: Use the API directly

### 1. Top up credits

Start a Stripe checkout session. Pay with any credit or debit card.

```bash
curl -X POST https://api.vidjutsu.ai/v1/api_keys \
  -H "Content-Type: application/json" \
  -d '{
    "credits": 100,
    "email": "you@example.com"
  }'
```

```json
{
  "url": "https://checkout.stripe.com/...",
  "sessionId": "cs_...",
  "credits": 100,
  "amount": 1000
}
```

Open the returned URL in your browser to complete payment.

### 2. Get your API key

After payment, poll the status endpoint:

```bash
curl "https://api.vidjutsu.ai/v1/api_keys/status?session=cs_..."
```

```json
{
  "status": "completed",
  "apiKey": "vj_live_abc123...",
  "clientId": "...",
  "balance": 100
}
```

<Warning>
Store your API key securely — it is shown once in this response.
</Warning>

### 3. Run a critic check (10 credits)

```bash
curl -X POST https://api.vidjutsu.ai/v1/critic \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: vj_live_abc123..." \
  -d '{
    "mediaUrl": "https://cdn.example.com/video.mp4",
    "mediaType": "video",
    "context": "fitness niche, talking-head format"
  }'
```

```json
{
  "score": 72,
  "issues": [
    "Audio levels inconsistent between 0:03 and 0:08",
    "Text overlay partially obscured at 0:12"
  ],
  "verdict": "fail"
}
```

### 4. Start a breakdown (10 credits)

```bash
curl -X POST https://api.vidjutsu.ai/v1/breakdown \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: vj_live_abc123..." \
  -d '{
    "mediaUrl": "https://cdn.example.com/video.mp4",
    "mediaType": "video",
    "prompt": "Analyze the hook and first 3 seconds"
  }'
```

```json
{
  "id": "va_abc123",
  "status": "processing"
}
```

Poll for the result:

```bash
curl "https://api.vidjutsu.ai/v1/breakdown?id=va_abc123" \
  -H "X-Api-Key: vj_live_abc123..."
```

### 5. Score viral potential (10 credits)

```bash
curl -X POST https://api.vidjutsu.ai/v1/score \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: vj_live_abc123..." \
  -d '{
    "mediaUrl": "https://cdn.example.com/video.mp4",
    "mediaType": "video",
    "content": "3 morning habits that changed my physique",
    "niches": ["fitness", "wellness"],
    "type": "talking-head"
  }'
```

```json
{
  "score": 84,
  "breakdown": {
    "hook": 92,
    "pacing": 78,
    "visualQuality": 88,
    "audioQuality": 80,
    "nicheRelevance": 82
  },
  "suggestions": [
    "Tighten the mid-section — pacing drops between 0:05 and 0:09",
    "Add a text overlay to reinforce the hook in the first second"
  ]
}
```
