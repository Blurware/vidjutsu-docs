## Download, transcribe, overlay

Three typed calls: pull a TikTok post into a stable VidJutsu asset, transcribe it with word-level timing, then burn a text overlay on the result.

```typescript
import { createClient } from "vidjutsu";

const vj = createClient();

// 1. Download the source video into a tenant-owned asset
const { data: download } = await vj.downloadTikTokVideo({
  url: "https://www.tiktok.com/@creator/video/123456789",
});

// 2. Transcribe it
const { data: transcript } = await vj.transcribeMedia({
  mediaUrl: download.videoUrl,
});

// 3. Burn an overlay using the first line of the transcript
const { data: overlay } = await vj.createOverlay({
  videoUrl: download.videoUrl,
  text: transcript.transcript.slice(0, 60),
  position: "bottom",
});

console.log(overlay.resultUrl);
```

## What happens

1. **Download** — the TikTok post becomes a stable, tenant-owned VidJutsu asset with a durable URL.
2. **Transcribe** — you get back the full transcript plus word-level timing.
3. **Overlay** — VidJutsu burns the text onto the video in TikTok-safe zones and returns the finished asset URL.
