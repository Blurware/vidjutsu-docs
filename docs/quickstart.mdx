## Option A: Use the CLI

The fastest way to get started. Install the CLI and authenticate in one step.

```bash
curl -fsSL https://vidjutsu.ai/install.sh | bash
```

Then authenticate with your card:

```bash
vidjutsu auth --card tok_live_your_card_token
```

Watch a video:

```bash
vidjutsu watch --video-url "https://cdn.example.com/video.mp4" --prompt "Is the hook effective?"
```

See the [CLI reference](/cli/index) for all commands.

## Option B: Use the API directly

### 1. Top up credits

Start a Stripe checkout session. Pay with any credit or debit card.

```bash
curl -X POST https://api.vidjutsu.ai/v1/api_keys \
  -H "Content-Type: application/json" \
  -d '{
    "credits": 100,
    "email": "you@example.com"
  }'
```

```json
{
  "url": "https://checkout.stripe.com/...",
  "sessionId": "cs_...",
  "credits": 100,
  "amount": 1000
}
```

Open the returned URL in your browser to complete payment.

### 2. Get your API key

After payment, poll the status endpoint:

```bash
curl "https://api.vidjutsu.ai/v1/api_keys/status?session=cs_..."
```

```json
{
  "status": "completed",
  "apiKey": "vj_live_abc123...",
  "clientId": "...",
  "balance": 100
}
```

<Warning>
Store your API key securely — it is shown once in this response.
</Warning>

### 3. Watch a video (10 credits)

```bash
curl -X POST https://api.vidjutsu.ai/v1/watch \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: vj_live_abc123..." \
  -d '{
    "videoUrl": "https://cdn.example.com/video.mp4",
    "prompt": "Is the hook strong enough for fitness TikTok?"
  }'
```

```json
{
  "response": "The hook lands in the first 1.2 seconds — direct address to camera with a bold claim. Strong for fitness TikTok. Consider adding a text overlay to reinforce for silent viewers."
}
```

### 4. Extract frames and audio (5 credits)

```bash
curl -X POST https://api.vidjutsu.ai/v1/extract \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: vj_live_abc123..." \
  -d '{
    "videoUrl": "https://cdn.example.com/video.mp4",
    "frames": "auto",
    "audio": true
  }'
```

```json
{
  "frames": 12,
  "frameUrls": ["https://cdn.vidjutsu.ai/..."],
  "audioUrl": "https://cdn.vidjutsu.ai/.../audio.wav",
  "duration": 14.2,
  "resolution": "1080x1920",
  "fps": 30
}
```

### 5. Transcribe speech (10 credits)

```bash
curl -X POST https://api.vidjutsu.ai/v1/transcribe \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: vj_live_abc123..." \
  -d '{
    "videoUrl": "https://cdn.example.com/video.mp4"
  }'
```

```json
{
  "words": 47,
  "duration": 14.2,
  "transcript": [
    { "word": "The", "start": 0.00, "end": 0.82 },
    { "word": "one", "start": 0.82, "end": 1.10 },
    { "word": "thing", "start": 1.10, "end": 1.40 }
  ]
}
```
