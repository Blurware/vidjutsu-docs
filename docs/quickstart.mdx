## Submit a brand meme task

Give VidJutsu a clear goal and a source video. The task records checkpoints while the private workflow imports the reference, checks cloneability, creates the character and starting image, generates motion, applies your wall text, and runs final QA.

```typescript
import { createClient } from "vidjutsu";

const client = createClient();

const task = await client.agentTasks.create(
  {
    prompt: "Clone this meme video for our brand with our character. Keep the timing and sound, then add the exact wall text: When our product finally clicks.",
    inputs: [
      {
        name: "source_video",
        url: "https://www.tiktok.com/@creator/video/123456789",
      },
    ],
  },
  { idempotencyKey: "brand-meme-001" },
);

for await (const event of client.agentTasks.tail(task.id)) {
  console.log(event.type, event.data);
}
```

## What happens next

1. **Import** — a TikTok or Instagram reference becomes a stable tenant-owned asset.
2. **Clone check** — the workflow stops immediately if the reference is a poor fit.
3. **Brand production** — VidJutsu creates the selected character, clean starting image, motion clone, and overlay.
4. **Final QA** — the workflow checks identity continuity, framing, duration, and the final wall text before returning the output.
