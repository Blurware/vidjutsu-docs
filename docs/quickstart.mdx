## Watch a video (10 credits)

Ask AI anything about a video. The response shape follows your prompt.

<Tabs>
  <Tab title="SDK">
    ```typescript
    import { createClient } from "vidjutsu";
    const vj = createClient();

    const { data } = await vj.watchMedia({
      mediaUrl: "https://cdn.example.com/video.mp4",
      prompt: "Is the hook strong enough for fitness TikTok?",
    });
    ```
  </Tab>
  <Tab title="CLI">
    ```bash
    vidjutsu watch \
      --mediaUrl "https://cdn.example.com/video.mp4" \
      --prompt "Is the hook strong enough for fitness TikTok?"
    ```
  </Tab>
  <Tab title="curl">
    ```bash
    curl -X POST https://api.vidjutsu.ai/v1/watch \
      -H "Content-Type: application/json" \
      -H "Authorization: Bearer $VIDJUTSU_API_KEY" \
      -d '{
        "mediaUrl": "https://cdn.example.com/video.mp4",
        "prompt": "Is the hook strong enough for fitness TikTok?"
      }'
    ```
  </Tab>
</Tabs>

```json
{
  "response": "The hook lands in the first 1.2 seconds — direct address to camera with a bold claim. Strong for fitness TikTok. Consider adding a text overlay to reinforce for silent viewers."
}
```

## Extract frames and audio (5 credits)

Pull frames, audio track, and metadata from any video.

<Tabs>
  <Tab title="SDK">
    ```typescript
    const { data } = await vj.extractMedia({
      mediaUrl: "https://cdn.example.com/video.mp4",
      frames: "auto",
      audio: true,
    });
    ```
  </Tab>
  <Tab title="CLI">
    ```bash
    vidjutsu extract \
      --mediaUrl "https://cdn.example.com/video.mp4" \
      --frames auto --audio
    ```
  </Tab>
  <Tab title="curl">
    ```bash
    curl -X POST https://api.vidjutsu.ai/v1/extract \
      -H "Content-Type: application/json" \
      -H "Authorization: Bearer $VIDJUTSU_API_KEY" \
      -d '{
        "mediaUrl": "https://cdn.example.com/video.mp4",
        "frames": "auto",
        "audio": true
      }'
    ```
  </Tab>
</Tabs>

```json
{
  "frames": [{ "index": 0, "url": "https://cdn.vidjutsu.ai/..." }],
  "audio": { "url": "https://cdn.vidjutsu.ai/...", "duration": 10.2 },
  "metadata": { "width": 1080, "height": 1920, "fps": 30, "duration": 10.0 }
}
```

## Transcribe speech (10 credits)

Speech-to-text with word-level timing.

<Tabs>
  <Tab title="SDK">
    ```typescript
    const { data } = await vj.transcribeMedia({
      mediaUrl: "https://cdn.example.com/video.mp4",
    });
    ```
  </Tab>
  <Tab title="CLI">
    ```bash
    vidjutsu transcribe \
      --mediaUrl "https://cdn.example.com/video.mp4"
    ```
  </Tab>
  <Tab title="curl">
    ```bash
    curl -X POST https://api.vidjutsu.ai/v1/transcribe \
      -H "Content-Type: application/json" \
      -H "Authorization: Bearer $VIDJUTSU_API_KEY" \
      -d '{
        "mediaUrl": "https://cdn.example.com/video.mp4"
      }'
    ```
  </Tab>
</Tabs>

```json
{
  "transcript": "the one thing nobody tells you about creatine",
  "words": [{ "word": "the", "start": 0.1, "end": 0.2 }],
  "language": "en",
  "duration": 10.2
}
```
