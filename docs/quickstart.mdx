## Option A: Use the CLI

The fastest way to get started. Install the CLI and authenticate in one step.

```bash
curl -fsSL https://vidjutsu.ai/install.sh | bash
```

Then authenticate with your card:

```bash
vidjutsu auth --card tok_live_your_card_token
```

Create a managed account:

```bash
vidjutsu account create --platform instagram --name "my_brand" --niche "fitness"
```

See the [CLI reference](/cli/index) for all commands.

## Option B: Use the API directly

### 1. Top up credits

Start a Stripe checkout session. Pay with any credit or debit card.

```bash
curl -X POST https://api.vidjutsu.ai/v1/api_keys \
  -H "Content-Type: application/json" \
  -d '{
    "credits": 100,
    "email": "you@example.com"
  }'
```

```json
{
  "url": "https://checkout.stripe.com/...",
  "sessionId": "cs_...",
  "credits": 100,
  "amount": 1000
}
```

Open the returned URL in your browser to complete payment.

### 2. Get your API key

After payment, poll the status endpoint:

```bash
curl "https://api.vidjutsu.ai/v1/api_keys/status?session=cs_..."
```

```json
{
  "status": "completed",
  "apiKey": "vj_live_abc123...",
  "clientId": "...",
  "balance": 100
}
```

<Warning>
Store your API key securely — it is shown once in this response.
</Warning>

### 3. Create a managed account

```bash
curl -X POST https://api.vidjutsu.ai/v1/accounts \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: vj_live_abc123..." \
  -d '{
    "platform": "instagram",
    "name": "my_brand",
    "niche": "fitness"
  }'
```

```json
{
  "id": "acc_abc123",
  "status": "creating",
  "creditsCharged": 32
}
```

### 4. Schedule a post

Once the account is active, submit a video for posting:

```bash
curl -X POST https://api.vidjutsu.ai/v1/posts \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: vj_live_abc123..." \
  -d '{
    "accountId": "acc_abc123",
    "videoUrl": "https://your-video-url.com/video.mp4",
    "caption": "Your caption here"
  }'
```
