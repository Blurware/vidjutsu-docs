The VidJutsu SDK is a typed TypeScript client generated from the [OpenAPI spec](https://raw.githubusercontent.com/tfcbot/vidjutsu-openapi/main/openapi/spec.json). It covers social import, scrape, watch, transcribe, overlay, disclaimer, compliance, and job polling.

## Install

```bash
npm install vidjutsu
```

## Quick start

```typescript
import { createClient } from "vidjutsu";

const vj = createClient();

const { data: download } = await vj.downloadTikTokVideo({
  url: sourceVideoUrl,
});

const { data: transcript } = await vj.transcribeMedia({
  mediaUrl: download.videoUrl,
});

console.log(transcript.transcript);
```

## Core methods

| Method | Description |
| --- | --- |
| `downloadTikTokVideo` / `downloadInstagramVideo` | Import a social reference into a stable asset. |
| `scrapeTikTokProfile` / `scrapeInstagramProfile` and related `scrape*` methods | Pull structured profile, post, comment, and search data. |
| `watchMedia` | Run prompted video or image understanding. |
| `transcribeMedia` | Speech-to-text with word-level timing. |
| `checkSpec` | Run a TOS/compliance check against configured rules. |
| `createOverlay` | Burn a text overlay onto a video. |
| `createDisclaimer` | Add a disclaimer card to a video. |
| `mediaJobs.get` / `awaitJob` | Poll a durable media job to completion. |

## Authentication

The client resolves your API key in order:

1. **Explicit** — `createClient({ apiKey: "vj_live_..." })`
2. **Environment variable** — `VIDJUTSU_API_KEY`
3. **Config file** — `~/.vidjutsu/config.json` (shared with the [CLI](/cli/auth))

```typescript
const vj = createClient();
```

Use a tenant-scoped key in a trusted environment. Do not place a key in a prompt, skill file, or public client bundle.
