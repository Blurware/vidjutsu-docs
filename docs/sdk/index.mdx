The VidJutsu SDK is a typed TypeScript client auto-generated from the [OpenAPI spec](https://raw.githubusercontent.com/tfcbot/vidjutsu-openapi/main/openapi/spec.json). Every method is fully typed — request bodies, response shapes, and query parameters.

## Install

```bash
npm install vidjutsu
```

## Quick start

```typescript
import { createClient } from "vidjutsu";

const vj = createClient();

// Watch a video (10 credits)
const { data } = await vj.watchMedia({
  mediaUrl: "https://cdn.example.com/video.mp4",
  prompt: "Is the hook strong enough for fitness TikTok?",
});
console.log(data.response);

// Extract frames and audio (5 credits)
const { data: extract } = await vj.extractMedia({
  mediaUrl: "https://cdn.example.com/video.mp4",
  frames: "auto",
  audio: true,
});

// Transcribe speech (10 credits)
const { data: transcript } = await vj.transcribeMedia({
  mediaUrl: "https://cdn.example.com/video.mp4",
});
console.log(transcript.transcript);
```

## Methods

| Method | Credits | Description |
|--------|---------|-------------|
| [`watchMedia`](/sdk/watch) | 10 | AI watches a video/image, answers your prompt |
| [`extractMedia`](/sdk/extract) | 5 | Extract frames, audio, and metadata |
| [`transcribeMedia`](/sdk/transcribe) | 10 | Speech-to-text with word-level timing |
| [`checkSpec`](/sdk/check) | 5 | Validate a VidLang spec against rules |
| [`createOverlay`](/sdk/overlay) | 5 | Burn text overlay onto video |
| [`uploadFile`](/sdk/upload) | 0 | Upload binary file to CDN |
| [`uploadFromUrl`](/sdk/upload) | 0 | Upload from URL to CDN |
| [`createAccount`](/sdk/accounts) / `listOrGetAccounts` / `updateAccount` / `deleteAccount` | 0 | Manage accounts |
| [`createPost`](/sdk/posts) / `listOrGetPosts` / `updatePost` / `deletePost` | 0 | Manage posts |
| [`createAsset`](/sdk/assets) / `listOrGetAssets` / `updateAsset` / `deleteAsset` | 0 | Manage assets |
| [`createReference`](/sdk/references) / `listOrGetReferences` / `updateReference` / `deleteReference` | 0 | Manage references |
| [`getBalance`](/sdk/balance) | 0 | Check credit balance |

## Authentication

The client resolves your API key in order:

1. **Explicit** — `createClient({ apiKey: "vj_live_..." })`
2. **Environment variable** — `VIDJUTSU_API_KEY`
3. **Config file** — `~/.vidjutsu/config.json` (shared with the [CLI](/cli/auth))

```typescript
// Reads from env var or config file automatically
const vj = createClient();

// Or pass explicitly
const vj = createClient({ apiKey: "vj_live_..." });

// Custom base URL (for staging, self-hosted, etc.)
const vj = createClient({ baseUrl: "https://staging.api.vidjutsu.ai" });
```

## Raw client

For endpoints not covered by convenience methods, or for advanced usage, access the raw `openapi-fetch` client:

```typescript
const { data, error } = await vj.api.GET("/v1/balance");
const { data, error } = await vj.api.POST("/v1/watch", {
  body: { mediaUrl: "...", prompt: "..." },
});
```
