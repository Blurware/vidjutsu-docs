The VidJutsu SDK is a typed TypeScript client generated from the [OpenAPI spec](https://raw.githubusercontent.com/tfcbot/vidjutsu-openapi/main/openapi/spec.json). It supports durable agent tasks as well as each atomic import, clone, overlay, watch, and job operation.

## Install

```bash
npm install vidjutsu
```

## Quick start

```typescript
import { createClient } from "vidjutsu";

const client = createClient();

const task = await client.agentTasks.create(
  {
    prompt: "Clone this meme video for our brand with our character and wall text.",
    inputs: [{ name: "source_video", url: sourceVideoUrl }],
  },
  { idempotencyKey: "brand-meme-001" },
);

for await (const event of client.agentTasks.tail(task.id)) {
  console.log(event.type, event.data);
}
```

## Core methods

| Method | Description |
| --- | --- |
| `agentTasks.create` / `agentTasks.tail` | Submit a goal-oriented workflow and tail durable events. |
| `downloadTikTokVideo` / `downloadInstagramVideo` | Import a social reference into a stable asset. |
| `clones.check` | Evaluate reference cloneability before generation. |
| `characters.create` | Create a reusable brand character. |
| `clones.createStartingImage` | Create a clean character-swapped starting frame. |
| `clones.createVideo` | Clone motion with Seedance or Kling motion control. |
| `createOverlay` | Apply final brand wall text. |
| `watchMedia` | Run prompted video or image understanding. |
| `awaitJob` / `mediaJobs.get` | Poll a durable media job. |

## Authentication

The client resolves your API key in order:

1. **Explicit** — `createClient({ apiKey: "vj_live_..." })`
2. **Environment variable** — `VIDJUTSU_API_KEY`
3. **Config file** — `~/.vidjutsu/config.json` (shared with the [CLI](/cli/auth))

```typescript
const client = createClient();
```

Use a tenant-scoped key in a trusted environment. Do not place a key in a prompt, skill file, or public client bundle.
