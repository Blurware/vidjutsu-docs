---
title: "cloneStartingImage"
description: "Composes the clean starting frame for a clone from a character and a prompt."
---

## Usage

```typescript
const { data, error } = await vj.cloneStartingImage({
  characterId: "char_abc123",
  firstFrame: "https://cdn.example.com/source-first-frame.jpg",
  prompt: "Character seated at a kitchen counter, holding the product",
});
```

## Parameters

Extract frame index `0` from the staged source first. Both that image and a stored character are required.

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `characterId` | `string` | Yes | A stored `char_...` id that supplies replacement identity |
| `firstFrame` | `string` | Yes | Public HTTPS source first-frame image whose composition is edited |
| `prompt` | `string` | Yes | How to compose the starting frame |

## Returns

| Field | Type | Description |
|-------|------|-------------|
| `imageUrl` | `string` | CDN URL of the starting frame |
| `model` | `string` | Model that produced the frame |

The returned frame is always clean, with no captions, watermarks, or platform UI.

## Errors

Returns 400 if `firstFrame` or `characterId` is missing or invalid, and 404 if `characterId` does not resolve to a character you own.

## Rate limit

Shares the daily clone admission limit. Resets at 00:00 UTC; exceeding it returns HTTP 429.
