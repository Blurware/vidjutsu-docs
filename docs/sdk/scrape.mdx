---
title: "scrape"
description: "Typed TikTok and Instagram import methods."
---

The SDK exposes TikTok and Instagram import methods only. Use the result as the input for downstream calls such as `watchMedia`, `transcribeMedia`, or `createOverlay`.

```typescript
import { createClient } from "vidjutsu";

const client = createClient();
const { data, error } = await client.scrapeInstagramPost({
  url: "https://www.instagram.com/reel/example/",
});

if (error) throw error;
```

Scrape methods return the provider's raw source media URLs and never stage media. To turn one video into a durable VidJutsu-hosted file, use the single-video download method, one video per call.

Available methods cover TikTok profiles, videos, transcripts, comments, user search, and trending; plus Instagram profiles, posts, reels, and comments. Every method uses the shared `scrape` rate group.

VidJutsu does not expose X, YouTube, or ad-library scrape methods.
