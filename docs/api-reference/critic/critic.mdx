## POST /v1/critic

Run a quality check on any video or image. Returns a score, a list of issues, and a pass/fail verdict. Use this as a gate before publishing AI-generated content.

### Request body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `mediaUrl` | string | Yes | Public URL of the video or image to evaluate |
| `mediaType` | string | Yes | `video` or `image` |
| `context` | string | No | Optional context about the content (e.g. niche, brand, intended audience) |

### Example request

```bash
curl -X POST https://api.vidjutsu.ai/v1/critic \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: vj_live_abc123..." \
  -d '{
    "mediaUrl": "https://cdn.example.com/video.mp4",
    "mediaType": "video",
    "context": "fitness niche, talking-head format"
  }'
```

### Example response

```json
{
  "score": 72,
  "issues": [
    "Audio levels inconsistent between 0:03 and 0:08",
    "Text overlay partially obscured at 0:12"
  ],
  "verdict": "fail"
}
```

### Response fields

| Field | Type | Description |
|-------|------|-------------|
| `score` | number | Quality score from 0 to 100 |
| `issues` | string[] | List of detected problems |
| `verdict` | string | `pass` or `fail` |
