---
title: "TikTok and Instagram imports"
description: "Supported social reference imports for the VidJutsu clone workflow."
---

VidJutsu supports public TikTok and Instagram reference imports only. Use the generated endpoint pages for each exact request and response contract.

Set `download_media` to `true` when the next clone stage needs VidJutsu-hosted media. After import, run clone check, create a character and starting image, submit the video clone, apply an overlay, and poll the job to completion.

Unsupported public surfaces: X/Twitter, YouTube, Google Ads, Meta Ads, LinkedIn Ads, and Reddit.
