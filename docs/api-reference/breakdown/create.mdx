## POST /v1/breakdown

Start a deep, structured breakdown of a video or image. This is an async operation — the response returns an ID you can poll with `GET /v1/breakdown`.

### Request body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `mediaUrl` | string | Yes | Public URL of the video or image to analyze |
| `mediaType` | string | Yes | `video` or `image` |
| `prompt` | string | No | Focus the analysis on a specific aspect (e.g. "hook effectiveness", "pacing", "visual quality") |

### Example request

```bash
curl -X POST https://api.vidjutsu.ai/v1/breakdown \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: vj_live_abc123..." \
  -d '{
    "mediaUrl": "https://cdn.example.com/video.mp4",
    "mediaType": "video",
    "prompt": "Analyze the hook and first 3 seconds"
  }'
```

### Example response

```json
{
  "id": "va_abc123",
  "status": "processing"
}
```

### Response fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Breakdown ID — use this to poll for the result |
| `status` | string | `processing` or `completed` |
