## GET /v1/posts

List all post records. Supports filtering by account and tags.

### Query parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `accountId` | string | Filter posts linked to a specific account |
| `tag.{key}` | string | Filter by tag value. Repeat for multiple tags. |

### Example request

```bash
curl "https://api.vidjutsu.ai/v1/posts" \
  -H "X-Api-Key: vj_live_abc123..."
```

### Filtering by account

```bash
curl "https://api.vidjutsu.ai/v1/posts?accountId=acc_8k2x" \
  -H "X-Api-Key: vj_live_abc123..."
```

### Filtering by tag

```bash
# All posts tagged with batch=week-12
curl "https://api.vidjutsu.ai/v1/posts?tag.batch=week-12" \
  -H "X-Api-Key: vj_live_abc123..."

# Combine filters
curl "https://api.vidjutsu.ai/v1/posts?accountId=acc_8k2x&tag.format=talking-head" \
  -H "X-Api-Key: vj_live_abc123..."
```

### Example response

```json
{
  "data": [
    {
      "id": "post_xyz789",
      "videoUrl": "https://cdn.example.com/video.mp4",
      "caption": "3 morning habits that changed my physique",
      "accountId": "acc_8k2x",
      "tags": [
        { "key": "batch", "value": "week-12" },
        { "key": "format", "value": "talking-head" }
      ]
    }
  ]
}
```
