## How it works

1. **Top up credits** — `POST /v1/api_keys` with your email. Pay with any card via Stripe.
2. **Get your API key** — Poll `GET /v1/api_keys/status` after payment. Your key is returned in the response.
3. **Make requests** — Pass your API key in the `X-Api-Key` header with every request.
4. **Poll for results** — Async endpoints return `202 Accepted` with a job ID. Use list endpoints to check status.
