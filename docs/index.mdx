---
title: "VidJutsu — Video Primitives for Agents"
description: "Deterministic video primitives your agent can call directly: scrape, download, watch, transcribe, check compliance, overlay, and distribute."
---

VidJutsu gives your agent a typed set of video primitives: pull social posts, download the media, transcribe or watch a video for what's in it, run a compliance check, burn an overlay or disclaimer, and hand the result off to distribution.

Each primitive is a single typed call with a durable job you can poll. No orchestration required on your side.

## MCP status

VidJutsu does **not** currently expose a public remote MCP server. Use the SDK, CLI, public skills, or agent authentication manifest today. A future MCP endpoint will be documented here when it is available.
