VidJutsu is an API and CLI for launching managed Instagram campaigns. Provision accounts, schedule posts, analyze performance, and drive traffic to any link — all from a single interface.

## Base URL

```
https://api.vidjutsu.ai
```

## How it works

1. **Top up credits** — `POST /v1/api_keys` with your email. Pay with any card via Stripe.
2. **Get your API key** — Poll `GET /v1/api_keys/status` after payment. Your key is returned in the response.
3. **Draft resources for free** — Create accounts and posts as drafts (0 credits). Edit them with PUT until ready.
4. **Provision and schedule** — Use command endpoints (`/provision`, `/schedule`) to charge credits and go live.
5. **Poll for results** — Async endpoints return `202 Accepted`. Use list endpoints to check status.

## Draft-first workflow

All resource creation is free. Credits are only charged at command endpoints:

| Action | Endpoint | Credits | Cost |
|--------|----------|---------|------|
| Create draft account | `POST /v1/accounts` | 0 | Free |
| Provision account | `POST /v1/accounts/provision` | 990 | $99.00 |
| Create draft post | `POST /v1/posts` | 0 | Free |
| Schedule post | `POST /v1/posts/schedule` | 36 | $3.60 |

Use `POST /v1/estimate` to preview costs before committing.
